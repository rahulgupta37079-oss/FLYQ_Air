## 🚀 DEPLOYMENT INSTRUCTIONS

### Your Local Machine (Windows)

**Step 1: Remove better-sqlite3**
```bash
cd C:\Users\PROFESSORHULK\FLYQ_Air\FLYQ_Air
npm uninstall better-sqlite3
del package-lock.json
rmdir /s /q node_modules
npm install
```

**Step 2: Build**
```bash
npm run build
```

**Step 3: Deploy**
```bash
npx wrangler pages deploy dist --project-name flyq-air
```

### OR: Use Cloudflare Dashboard (Easier!)

1. Go to https://dash.cloudflare.com/
2. Workers & Pages → flyq-air → Settings
3. Builds & deployments → Connect to Git
4. Select `rahulgupta37079-oss/FLYQ_Air` → Branch `main`
5. Build command: `npm run build`
6. Build output: `dist`
7. Click "Save and Deploy"
