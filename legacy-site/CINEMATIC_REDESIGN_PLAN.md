# FLYQ Cinematic Redesign Plan

## Reference: https://deft-syrniki-e62766.netlify.app/

## Key Design Elements to Implement:

### 1. **Hero Section** (Full-screen video)
- Black background with full-screen video
- Gradient overlays (top-to-bottom black fade)
- Floating UI elements (status badge, navigation pills)
- Huge typography with transparent gradient text
- Bottom-aligned heading with subtitle
- CTA buttons at bottom

### 2. **Feature Showcase Sections**
- Multiple full-height (110vh) video sections
- Each section has:
  - Background video (50% opacity)
  - Gradient overlays (top & bottom fade to black)
  - Large heading with gradient text
  - Subtitle description
  - Text alignment varies (center, left, right)

### 3. **Typography**
- **Huge headings**: 6xl to 9xl
- **Gradient text**: from-white to-white/40
- **Tracking**: tighter for headings
- **Drop shadows**: on large text

### 4. **Colors**
- **Background**: Pure black
- **Accents**: White, Blue (exec-blue)
- **Text**: White primary, gray-300/400 secondary
- **Borders**: white/10 to white/20 opacity

### 5. **Components**
- Rounded pill navigation
- Status badges with live indicators
- Floating info cards
- Spec cards in grid
- Pricing tiers
- CTA section with large rounded container

### 6. **Animations**
- Fade in on scroll
- Transform translateY
- Opacity transitions
- Scale effects

## Implementation Steps:

1. Create new hero with full-screen video + floating UI
2. Add feature showcase sections (4-5 sections)
3. Add specs grid section
4. Add pricing/CTA section
5. Style navigation bar
6. Add animations with CSS classes

## Timeline: ~30 minutes to implement
