## 🎭 ANIMATION SYSTEM

### Custom Animations
```css
@keyframes float - Vertical floating (3s)
@keyframes float-slow - Slow floating with rotation (6s)
@keyframes pulse-slow - Opacity + scale pulse (4s)
@keyframes fadeInUp - Fade in from bottom
@keyframes shimmer - Shimmer loading effect
@keyframes glow-pulse - Shadow pulsing
@keyframes gradient-shift - Background gradient animation
@keyframes bounce-in - Scale bounce entrance
```

### Utility Classes
- `.animate-float` - Apply floating animation
- `.animate-float-slow` - Slow floating
- `.animate-pulse-slow` - Slow pulse
- `.animate-fadeInUp` - Fade in entrance
- `.animate-glow-pulse` - Glowing pulse
- `.animate-gradient-shift` - Gradient animation
- `.glass-card` - Glass morphism effect
- `.hover-lift` - Lift on hover
- `.gradient-text-orange` - Orange gradient text
- `.gradient-text-blue` - Blue gradient text
